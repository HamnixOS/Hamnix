User: *"Your time constraints, scope constraints tend to be hurtful not helpful. It makes the agents afraid to code. Let him run the fuck wild."*

**Drop from briefs:** time budgets, file-scope lists, single-commit mandates, "stop on architectural blockers," collision avoidance hand-holding, "be careful about Z" caution-induction.

**Keep in briefs:** don't edit README/TODO/STATUS/LANGUAGE/MEMORY/CLAUDE; no `git restore`/`reset --hard`/`clean -fd`; don't trust own PASS; architectural facts (Plan 9, no sockets, ...); named primitives ("`udp_send` from `drivers/net/udp.ad`"); the mission and what observable proves it.

**Replace constraint language with empowerment:** "fix it wherever it lives," "go deep," "report honestly including dead ends," "the orchestrator handles merges."

**Brief shape:** goal → where the code lives → architectural facts → success markers → "code as long as you need."
