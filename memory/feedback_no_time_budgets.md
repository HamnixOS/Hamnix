User: *"Stop giving the agents a time budget. They can code for as long as needed."*

Time budgets push toward premature stubs, distort honest reporting ("I ran out of time" vs "I chose the wrong abstraction"), and convey false anxiety agents pattern-match against. The agent decides pacing. Six hours of real engineering beats three of stubs.

Superseded in scope by [[feedback-let-agents-run-wild]] which extends this to *all* over-constraint.
