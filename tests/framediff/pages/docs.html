Getting started

This page walks through installing Hamnix and bringing up the desktop for the first time. Each section builds on the previous one.

Install

Write the live image to a USB stick and boot it. Run install to copy the root filesystem to your NVMe drive and reboot.

First boot

The desktop starts automatically. Open a terminal with the launcher and run hamsh to get an interactive shell.
